### Technology Used
- **Swift** - Core language
- **SwiftUI** - UI framework
- **URLSession** - Networking
- **File Manager** - Persisting Recipes

### How to Run
1. Clone the repository.
```
git clone https://github.com/DTeakell/Recipes.git
```
2. Open in Xcode.
3. Build and run on an iPhone simulator or physical device.

### Requirements
- **Xcode 15 +**
- **iOS 16 +**
